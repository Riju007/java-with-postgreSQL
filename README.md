# java-with-postgreSQL
How to connect with PostgreSQL with java and storing the data in a ArrayList
Requirements:
	1. java / jdk (JDK 1.8 higher recommended)
	2. eclipse
	3. PostgreSQL and pgAdmin(GUI for postgreSQL)
	4. sample database DVD Rental. Download it from - http://www.postgresqltutorial.com/postgresql-sample-database/
	   Unzip the tar file and load in postgreSQL
	5. Download required jar from - https://jdbc.postgresql.org/download.html
	6. Load the jar file in classpath
